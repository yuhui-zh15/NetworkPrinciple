# server文件夹为服务器
编译方式：内含makefile文件，终端输入make即可
运行方式：终端输入./server 5000，请使用5000号端口

# client文件夹为客户端
## qtclient文件夹为Qt实现的客户端
编译方式：Qt 5.6及以上版本编译
运行方式：运行Qt编译后文件
APP：提供了macOS下macdeplotqt发布后的版本，可以直接运行tcpclient.app
## pyclient文件夹为Python3实现的简易客户端（不支持文件传输）
编译方式：无需编译
运行方式：终端输入python3 client.py --port=5000
